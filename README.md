## 系统环境

兼容 macOS 和 Linux。
Windows建议使用win10的Linux Subsystem进行开发。 

Linux需要安装uuidgen命令与zip命令。 

## 用法: 


创建插件
````
-c/--create project_name
例: ./cpkgen.sh -c myProj
````


打包插件
````
-p/--package project_name
例: ./cpkgen.sh -p myProj.cplugin
````
